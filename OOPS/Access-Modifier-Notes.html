Access Modifiers in Java

In Java, there are four types of access modifiers, which control the visibility of classes, methods, and variables. Hereâ€™s a detailed explanation of each:

1. Default (Package-Private) Access Modifier

Scope: Accessible only within the same package. It is the default access level if no access modifier is explicitly specified.

Usage: Suitable for classes and members meant to be shared only within the package. 

class DefaultClass { // No modifier means 'default'
    void display() {
        System.out.println("Default Access Modifier");
    }
}

2. Public Access Modifier

Scope: Accessible from anywhere, both within the same package and outside the package.

Usage: Typically used for classes, methods, and variables that need to be accessed widely. 

public class PublicClass {
    public void display() {
        System.out.println("Public Access Modifier");
    }
}

3. Private Access Modifier

Scope: Accessible only within the same class.

Usage: Often used for data encapsulation, hiding the internal details of a class.

Note: To allow controlled access to private members, getters and setters are often used. These methods provide a way to read or update private fields while maintaining control over how the fields are accessed or modified. 

class PrivateClass {
    private int value;

    // Getter method
    public int getValue() {
        return value;
    }

    // Setter method
    public void setValue(int value) {
        if (value > 0) { // Validation logic can be added
            this.value = value;
        }
    }

    private void display() {
        System.out.println("Private Access Modifier");
    }

    public void show() {
        display(); // Accessible within the same class
    }
}

4. Protected Access Modifier

Scope: Accessible within the same package and by subclasses in different packages.

Usage: Useful for inheritance, as it allows child classes to access parent class members. 

class Parent {
    protected void display() {
        System.out.println("Protected Access Modifier");
    }
}

public class Child extends Parent {
    public static void main(String[] args) {
        Child obj = new Child();
        obj.display(); // Accessible because Child extends Parent
    }
}

Full Code Demonstration

// Default access modifier
class DefaultClass {
    void display() {
        System.out.println("Default Access Modifier");
    }
}

// Public access modifier
public class PublicClass {
    public void display() {
        System.out.println("Public Access Modifier");
    }
}

// Private access modifier
class PrivateClass {
    private int value;

    // Getter method
    public int getValue() {
        return value;
    }

    // Setter method
    public void setValue(int value) {
        if (value > 0) {
            this.value = value;
        }
    }

    private void display() {
        System.out.println("Private Access Modifier");
    }

    public void show() {
        display(); // Accessed within the same class
    }
}

// Protected access modifier
class Parent {
    protected void display() {
        System.out.println("Protected Access Modifier");
    }
}

public class AccessModifiersDemo extends Parent {
    public static void main(String[] args) {
        // Default access
        DefaultClass defaultObj = new DefaultClass();
        defaultObj.display();

        // Public access
        PublicClass publicObj = new PublicClass();
        publicObj.display();

        // Private access with getter and setter
        PrivateClass privateObj = new PrivateClass();
        privateObj.setValue(10);
        System.out.println("Value: " + privateObj.getValue());
        privateObj.show();

        // Protected access
        AccessModifiersDemo protectedObj = new AccessModifiersDemo();
        protectedObj.display(); // Accessed due to inheritance
    }
}
